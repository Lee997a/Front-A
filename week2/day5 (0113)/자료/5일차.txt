
5차시
 
1. 요소(태그)의  display속성을 구분하고 특징을 알 수 있다 (block, inline-block, inline)
2. 시멘틱 태그를 사용할 수 있다
3. 리뷰만들기 
4. position:absolute  이해하기 ( 별붙이기, 갤러리화면 만들기)
5. 메뉴만들기  1단, 2단 (하위메뉴)
6. 장바구니 만들기 



1. block 요소
 - 항상 새로운 줄에서 시작 / 전체 가로 너비를 차지 / 기본적으로 부모 요소의 너비으 100%를 차지 / 너비와 높이를 설정할 수 있다.
1) div
2) p
3) hn 태그
4) 목록태그(ul, ol. li)
5) table
6) section


2. inline 요소
 - 같은 줄에서 다른 요소들과 나란히 배치 / 콘텐츠의 크기만큼 너비와 높이를 차지 / 기본적으로 너비와 높이를 설정할 수 없음(크기를 설정할 수 없음)
1) span
2) a
3) strong
4) em
5) b
6) i


3. inline-block 요소
 - inline 처럼 같은 줄에 나란히 배치 가능 / block 처럼 너비와 높이 설정 가능 / inline 요소의 특성과 block 요소의 특성을 결합.

1) img
2) button
3) input
