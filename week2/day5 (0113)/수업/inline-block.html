inline-block 태그

인라인의 속성을 갖으면서 크기를 가질 수 있다

inline-block 종류

  • <img>: 이미지를 나타냅니다.
  • <input>: 사용자 입력 필드를 제공합니다.
  • <button>: 클릭 가능한 버튼입니다.
  • <select>: 드롭다운 리스트를 제공합니다.
  • <textarea>: 여러 줄의 입력 필드입니다.

<는 html 엔티티라고 불린다
html에서 특정문자 (<, >, &)는 태그나 특수기호로 인식될 수 있기 때문에 안전하게 출력하려면 html 엔티티를 사용하면 된다.
